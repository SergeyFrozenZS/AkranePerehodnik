# Site-Perehodnik

Простая структура для одностраничного сайта-переходника.

## Файлы

- `index.html` — главная страница
- `css/style.css` — стили проекта
- `js/script.js` — логика вставки ссылок

## Как использовать

1. Открой `index.html` в браузере.
2. Заменяй placeholder-ссылки в `js/script.js`:
   - `GITHUB_URL`
   - `KWORK_URL`
   - `TELEGRAM_URL`
   - `TELEGRAM_CHANNEL_URL`
   - `INSTAGRAM_URL`
3. При необходимости меняй имя и описание в `index.html`.

## Развертывание

Файл `index.html` готов к загрузке на Vercel, GitHub Pages или любой статический хостинг.
